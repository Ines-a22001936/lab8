Board Game - Chess
This was the first project I did as a group for the Fundamentos de Programação's project (1st semester of the 1st year).
The program, developed in Kotlin, is a version of the well-known “Board Game - Chess”, in which most of the rules of the traditional game are followed.
With that being said, this game gives users the possibility to choose the dimensions of the board as well as the possibility of being able to see the pieces.
The game begins!, in which each piece has its own movement on the board.
The player who manages to "eat" all the opponent's pieces wins!

Zodiac
This project was developed for the Linguagens de Programação's discipline (2nd semester of the 1st year).
This program (C language) only needs the date of birth and the current date, and easily determines the user's age and sign.
In addition, the program also determines whether the person is fit to work or is eligible for retirement.
